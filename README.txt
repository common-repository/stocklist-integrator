Add a stocklist to your website. Works with multiple providers (Autotelex, Hexon, RDC / InMotiv, UCC, VWE)

== Description ==
Easily integrate a stocklist to your Wordpress website. Simply choose your provider and add the following shortcode

`[stocklist code="abcd"]`
